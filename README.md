Tetrissing
==========

Tetris clone in Processing

Keys:
 - Arrow keys to transform piece
 - G,F,K,M - toggle some features
 - P for pause
 - Space to immediately drop piece

To Build:  
 $ make  
 $ python -m SimpleHTTPServer  
Then go to http://localhost:8080
